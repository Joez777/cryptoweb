# Crypto Exchange API

ระบบ API สำหรับแลกเปลี่ยนสกุลเงินคริปโต

## การติดตั้ง

1. Clone repository:
   ```bash
   git clone <repository-url>
   cd crypto-exchange
   ```

2. ติดตั้ง dependencies:
   ```bash
   npm install
   ```

3. รีเซ็ตฐานข้อมูล:
   ```bash
   node models/index.js
   ```

4. สร้างข้อมูลตัวอย่าง:
   ```bash
   node seeders/demo.js
   ```

5. รันเซิร์ฟเวอร์:
   ```bash
   npm start
   ```
